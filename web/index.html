Rust WASM RustRadio Example

Instructions

First ensure WASM is fully loaded. The buttons should become enabled, and a massage saying "WASM loaded" should show up.

You can click the "add" button to run a simple test, making sure that WASM works.

Then you can upload a 50ksps 32bit complex float I/Q via the browse button. Rustradio will run on it, and will give you the bytes for the first 1200bps AX.25 packet decoded.
